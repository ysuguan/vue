## CLI操作入门

</summary>

### 安装
+ 全局安装vue-cli3：
  ```
  npm install -g @vue/cli
  ```
+ 在3的基础上安装2的模板(在这个基础上就可以选用2)
  ```
  npm install @vue/cli-init -g
  ```

### cli2使用
+ cli2初始化：
  ```
  vue init webpack 项目名称
  ```

### 关闭eslint
+ /config/index.js useEslint: false

### cli3使用
+ cli3初始化：
  ```
    vue create 项目名称
  ```
+ 打开vue图形管理界面
  ```
    vue ui
  ```
